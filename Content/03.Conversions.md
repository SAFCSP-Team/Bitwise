# Conversions 
<!-- Why is it important to know how to convert between number systems? -->
Converting between number systems is an essential skill when working with low-level computing and programming. Computers internally operate in binary, while humans commonly use decimal, and programmers often work with hexadecimal or octal. Because of this difference, understanding how to convert between number systems helps bridge the gap between how computers store data and how humans read and interpret it.

Being able to convert between binary, decimal, octal, and hexadecimal allows you to understand memory values, debug programs, interpret machine-level data, and work more effectively with low-level systems.

## Binary ↔ Decimal

To convert a binary number (base-2) to decimal (base-10), multiply each bit by \(2^{position}\), starting from the rightmost bit at position \(0\). Then sum the results.

Example:
```
1011₂

(1 × 2³) + (0 × 2²) + (1 × 2¹) + (1 × 2⁰)
= 8 + 0 + 2 + 1
= 11₁₀
```
Another way:
```
Binary:     1   0   1   1
Power:      2³  2²  2¹  2⁰
Value:      8   0   2   1
```
Add the values of the bits that are equal to 1, and ignore the bits that are 0.
```
Total = 11
```


To convert a decimal number (base-10) to binary (base-2), repeatedly divide the number by 2 and record the remainder.  
Continue dividing the quotient by 2 until the result becomes 0.


Example:
```
11 ÷ 2 = 5 remainder 1
5 ÷ 2 = 2 remainder 1
2 ÷ 2 = 1 remainder 0
1 ÷ 2 = 0 remainder 1

Result:1011
```
The binary number is obtained by reading the remainders **Reading the remainders from bottom to top**.


## Binary ↔ Hexadecimal

To convert a binary number to hexadecimal, group the binary digits into sets of four bits (called **quads**) starting from the right. Then replace each group with its hexadecimal equivalent.

Example:
```
Binary:    10101111
Grouped:   1010 1111
Hexadecimal: A    F

Result: 10101111₂ = AF₁₆
```
To convert a hexadecimal number to binary, simply reverse the process by replacing each hexadecimal digit with its 4-bit binary equivalent.


## Binary ↔ Octal

To convert a binary number to octal, group the binary digits into sets of three starting from the right (least significant bit).  
If necessary, add leading zeros on the left to complete the groups. Then convert each 3-bit group to its equivalent octal digit.

> Each octal digit represents **exactly three binary bits**.

Example:
```
Binary:      1101
Pad:       001 101
Octal:       1   5

Result: 1101₂ = 15₈
```

To convert an octal number to binary, perform the reverse process.


**Now Binary acts as a central bridge between number systems.**

For example, if you want to convert from hexadecimal to decimal, you can use binary as an intermediate step:

Hexadecimal ↔ Binary ↔ Decimal  <br> OR <br>  Hexadecimal ↔ Binary ↔ Octal

But direct conversion methods between other number systems are also possible and sometimes more efficient.

## Decimal → Hexadecimal
To convert a decimal number to hexadecimal, repeatedly divide the number by **16** and record the remainders.  
Continue dividing the quotient by 16 until the result becomes **0**.  
Read the remainders **from bottom to top** to obtain the hexadecimal number.

> Remainders **10–15** are represented by the letters **A–F**.

Example:
```
345 ÷ 16 = 21 remainder 9
21 ÷ 16 = 1 remainder 5
1 ÷ 16 = 0 remainder 1

Result: 345₁₀ = 159₁₆
```
**Reading the remainders from bottom to top**


## Hexadecimal → Decimal


To convert a hexadecimal number (base-16) to decimal (base-10), multiply each digit by **16 raised to the power of its position**, starting from 16⁰ on the right and increasing to the left. Then add all the results together.

> In hexadecimal, the letters represent values:
> A = 10, B = 11, C = 12, D = 13, E = 14, F = 15.

Example:
```
159₁₆

(1 × 16²) + (5 × 16¹) + (9 × 16⁰)
= 256 + 80 + 9
= 345₁₀
```


## Decimal → Octal


To convert a decimal number to octal, repeatedly divide the number by **8** and record the remainders.  
Continue dividing the quotient by 8 until the result becomes **0**.  

Example:
```
125 ÷ 8 = 15 remainder 5
15 ÷ 8 = 1 remainder 7
1 ÷ 8 = 0 remainder 1

Result: 125₁₀ = 175₈
```

**Reading the remainders from bottom to top**

## Octal → Decimal
To convert an octal number to decimal, multiply each digit by **8 raised to the power of its position**, starting from 8⁰ on the right and increasing to the left. Then add the results together.

Example:
 ```
175₈

(1 × 8²) + (7 × 8¹) + (5 × 8⁰)
= 64 + 56 + 5
= 125₁₀
 ```
