## Encoding

Computers store and process all data in binary form (0s and 1s).  
However, humans work with characters such as letters, numbers, and symbols.  
**Encoding** is the method used to represent these characters as binary values so that computers can store, transmit, and process them.

Different encoding systems assign specific binary numbers to characters.  
Some common character encoding standards include **ASCII** and **Unicode**.

### ASCII

ASCII (American Standard Code for Information Interchange) is one of the earliest character encoding standards.  
It represents characters using **7 bits**, allowing a total of **128 different characters**, including letters, digits, punctuation, and control characters.

Example:

'A' → 65 → 01000001
'a' → 97 → 01100001
'0' → 48 → 00110000

### Unicode

Unicode is a modern encoding standard designed to represent characters from **many languages and writing systems** around the world.

Unlike ASCII, Unicode supports **thousands of characters**, including emojis and characters from different alphabets.

Common Unicode encodings include:

- UTF-8
- UTF-16
- UTF-32


> [!NOTE]
> ASCII is actually a subset of Unicode.  
> This means that standard ASCII characters have the same binary representation in UTF-8.
