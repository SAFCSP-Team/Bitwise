
# Introduction

Bitwise operations form one of the fundamental building blocks of computer concepts.  
Understanding bitwise operations means understanding how data is actually manipulated at the lowest level inside a computer.

To understand bitwise operations, you first need to understand what a **bit** is (short for *binary digit*).  
To understand what a bit is, you need to understand the **binary number system**.  
To truly understand the binary system, it is helpful to relate it to a familiar number system, which is the **decimal system (base 10)**. 

<br>


> All data in a computer is ultimately represented in binary as sequences of 0s and 1s. Whether it is text, images, sound, video, code, or memory addresses, everything is stored as numbers in binary form. This is why the binary number system forms the foundation of computing..

Software developers, network analysts, and computer security analysts are some of the people who need to deeply understand binary.  
A software developer needs to understand binary to recognize the limitations of a computer when performing arithmetic and to write code that accounts for those limitations.  
A network analyst uses binary to divide and configure networks using subnet masks.  
A security analyst uses binary to understand data encryption and analyze security attacks.

Some resources explain binary as a process of conversion. For example, to convert from the decimal number system (base 10) to binary:

1. Repeatedly divide the decimal number by 2, and continue dividing the quotient by 2 until the quotient becomes 0.  
2. Write the remainders in reverse order; this sequence represents the binary number.

Example:

```
13 ÷ 2 = 6 R1
6 ÷ 2 = 3  R0
3 ÷ 2 = 1  R1
1 ÷ 2 = 0  R1

Binary = 1101₂

```


However, this is not truly an explanation of binary; it is only a rote process for converting from decimal to binary. It does not help in deeply understanding binary.

A better approach to understanding binary is to explore its importance in computing and how it relates to other number systems. After building this foundation, we can then perform operations at the binary level, and this is what this repository covers.
