## Numbering System
A number system is a mathematical framework used to represent numbers through a defined set of symbols and rules. It allows quantities and values to be expressed in different forms.

Number systems provide a structured and consistent way to represent numbers, enabling arithmetic operations and ensuring clear communication in both mathematics and computer systems.


<br>

> [!NOTE]
> All data in a computer is ultimately represented in binary as sequences of 0s and 1s. Whether it is text, images, sound, video, code, or memory addresses, everything is stored numerically. This is why number systems form the core of computing.

### Binary
The binary number system is the fundamental representation of all data inside computers.  
It uses only two digits: 0 and 1. Every value in a computer—numbers, text, images, and instructions—is ultimately stored in binary form.
Binary can be long for large amount of data, so there are more systems like octal and hexadecimal that allow binary sequence to be broken into groups making it less error-prone and readable compared to binary.

### Decimal
The decimal number system is the standard counting system used by humans in daily life.  
It is base 10 and uses digits from 0 to 9.

Computers internally operate using binary (base 2), but they often convert binary values into decimal form to make them easier for humans to read and understand. In simple terms, humans naturally understand decimal, while computers fundamentally operate in binary.
<br>

### Octal

The octal number system is a base-8 system that uses digits from 0 to 7.  
It provides a more compact way to represent binary numbers because each octal digit corresponds to three binary bits.

Octal is commonly used in computing when working with grouped binary values. A practical example is file permissions in Unix/Linux systems, where permissions such as read, write, and execute are represented using octal numbers (e.g., 755).

### Hexadecimal
The hexadecimal number system is base 16 and uses digits from 0–9 and letters A–F.  
It is widely used in programming and computer systems because it provides a compact and readable representation of binary data. One hexadecimal digit represents four binary bits.

### Base N

A base-N number system is a generalized numbering system where the base (N) can be any positive integer.  
By choosing a base and a set of symbols, it is possible to create different number systems. Binary (base 2), octal (base 8), decimal (base 10), and hexadecimal (base 16) are all examples of base-N systems.

In any base-N system, each position represents a power of the base, and the value of a number is calculated based on those positional weights.
